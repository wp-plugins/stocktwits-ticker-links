== Description ==

This plugin changes your standard ticker symbols like $AAPL or $GOOG and will automatically link the ticker to realtime discussions at stocktwits.com

It's pretty simple to run, just drop it in the plugin directory and activate that's all it takes.

== Installation ==

1. Upload `stocktwits_ticker_links.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Sit back and write posts about your stocks using the $TICKER format and all will be linked to what people are saying about it on stocktwits.com
